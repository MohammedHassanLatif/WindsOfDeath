Game Design

Game Overview

'Winds of Death' is a top-down 2D co-op shooter set in a zombie apocalypse. The players' objective is to escape a sequence of diverse levels while killing as many zombies as possible. They will be able to pick up various weapons such as a pistol, assault rifle, machine gun and grenades. There will also be a competitive scoring system to encourage competition and increase replayability.

Inspirations

The design of the game was inspired by top-down shooters from years past. These include Boxhead: 2Play Rooms, Zombies Ate My Pizza, and Mecha Team Maverick. We found games like these enjoyable because of their high replayability and challenge. Mechanics they have in common are a level progression system, scoring system and upgrade system for the players abilities. We seek to put our own spin on the top-down style as well as replicate and enhance the enjoyable and addictive gameplay loop.

Development

In the development of our game, we will use the Agile development methodology. The essence of Agile is making the product in small iterations, each of which inform the planning for the project going forward. We decided that this suits the project best as it allows us to be flexible with how we develop the game. For version control we will be using Git. This will allow us to keep all of our code in sync. We've all had code-related problems in the past so this extra safety net will help the project run smoother.
